noun.wtf v2 API

Compact endpoint for reading the current auction state from Ethereum mainnet.

Project Context

This endpoint is part of the xpp noun.wtf hackathon grant. Detailed code, firmware notes, and step-by-step DIY instructions are available in the nounwtf-xpp GitHub repository.

GET/api/auction

Calls the contract's read-only auction() function and returns JSON.

Response

timeLeft is calculated from endTime and returned as HH:MM.

{
  "success": true,
  "data": {
    "nounId": "52",
    "amount": "0",
    "amountETH": "0",
    "startTimeISO": "2026-06-30T23:36:35.000Z",
    "endTimeISO": "2026-07-01T23:36:35.000Z",
    "timeLeft": "00:00",
    "bidder": "0x0000000000000000000000000000000000000000",
    "settled": false
  }
}

Errors

RPC failures return HTTP 500 with { "success": false, "error": "..." }.
