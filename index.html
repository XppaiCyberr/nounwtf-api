noun.wtf API Endpoints

Compact endpoints for reading current auction state from Ethereum mainnet.

Project Context

This endpoint is part of the xpp noun.wtf hackathon grant. Detailed code, firmware notes, and step-by-step DIY instructions are available in the nounwtf-xpp GitHub repository.

GET/api/grant

Latest non-canceled noun.wtf v2 grant endpoint.

Open /api/grant

Source
https://spirited-flexibility-production-3c30.up.railway.app/graphql
Query
grants(limit: 1, orderDirection: "DESC", where: {status_not: CANCELED})
CORS
Enabled for browser clients.

Response

timeLeft is calculated from endTime and returned as HH:MM.

bidder is ENS-resolved when available. bidderAddress keeps the raw address. The zero address is returned as null.

{
  "success": true,
  "data": {
    "nounId": "52",
    "amount": "0",
    "amountETH": "0",
    "startTime": "1782862595",
    "endTime": "1782948995",
    "timeLeft": "00:00",
    "bidder": null,
    "bidderAddress": null,
    "settled": false
  }
}

Errors

RPC failures return HTTP 500 with { "success": false, "error": "..." }.
